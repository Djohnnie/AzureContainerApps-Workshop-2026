# Chapter 02 — Deploying to Azure Container Apps

## What you'll learn

- Azure Container Apps environments and resource model
- External ingress: custom domains, TLS, traffic rules

## Lab

In the lab for this chapter you will deploy the Snake frontend container you built in Chapter 01 to a real Azure Container Apps environment, configure a public ingress URL, and verify it scales to zero after a period of inactivity.
