# Chapter 05 — Scaling & State Management

- Horizontal scaling of Orleans to multiple silos

In the lab for this chapter you will scale the Orleans cluster to multiple replicas, configure Azure Table Storage as the clustering back-end, and observe grain activations moving between silos as you scale up and down.
